Project Name :

Snake Game

  • This Project is about a snake chasing apple

  • In this Game the snake size was fixed at the start and will be increased as it eats apples

  • The Game ends when the snake bites it self

  • With the increase in the size, the speed of the snake also increases

The Technologies :

Java

  • This game is devoloped in Java and the IDE is Eclipse

  • Java is Programming language which is secure robust and user friendly

  • Java is a Object Oriented Language in which the complex behavior of some entity will be represented by the class and the instance of the particular class will be an object

  • In this project I've used Jpanel and some action listeners to make the snake move
