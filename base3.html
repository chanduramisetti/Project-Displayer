Project Name :

Automatic Door control using Ultrosonic Sensor

  • This Project is to control a door whne ever a person approches the door

  • In this when ever a person is approaching the door the Ultrosonic Sensor detects that and opens the door and after few seconds when there is no person approching it closes

The Technologies :

Embedded C

  • This Project is devoloped using Arduino Uno

  • The Hard Ware are

    • Ultra-Sonic Sensor

    • Buzzer

    • Arduino Uno

    • Ada Fruit Backlight LCD

    • Motor controller with L293D

  • The working will be, when ever a person is approaching the door the Ultrosonic Sensor detects that and opens the door and after few seconds when there is no person approching it closes

  • The project was done in Arduino IDE with embedded c
