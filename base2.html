Project Name :

Remote Range Meter

  • This Project is about sencing the distance of an object

  • In this when ever an object is placed in front of the unit it will display the distance and upload it into the cloud so the end user can know it

The Technologies :

Embedded C

  • This Project is devoloped GSIMO-V board

  • The Hard Ware are

    • Ultra-Sonic Sensor

    • Ada Fruit OLED display

    • GSIMO-V Board

    • ESP-32, 32-bit micro-controller

  • The working will be, when ever a body is placed in front of the unit the distance will be updated on the LED and also on the remote mobile the user is using

  • The App was built in Kodular rapid mobile app which has two modes Developer and Block Modes

  • The cloud used was non-sql Google fire Base

  • The unit was connected to the wifi router as the Esp-32 has on chip wifi-module
